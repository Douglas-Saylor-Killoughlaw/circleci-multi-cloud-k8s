Project Info

This website is a simple Java Spring Boot application packaged by Google's Jib. We deployed the containers to GCP and AWS Kubernetes clusters using CircleCI. Global load balancing is provided by CloudFlare. It is a demonstration project for CircleCI's blog series on multi-cloud container deployments.

You can see the full sample code and configuration for deploying Java applications to multiple cloud providrs using Docker and Kubernetes in our Github repository. If you have any feedback or questions, please contact @edwardawebb.

Happy Shipping!
